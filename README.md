# 💪 Gym Bros Fit

A full-stack fitness tracking application that allows users to set goals, track progress, and stay consistent with their health journey.

---

## 🚀 Live Demo

🌐 https://gymbrosfit.netlify.app

---

## 📦 Repositories

* 🎨 Frontend: https://github.com/farnoushdf/gym-bros-frontend-v2
* ⚙️ Backend: https://github.com/farnoushdf/gym-bros-backend-v2

---

## 📌 Features

* 🔐 Authentication (Signup / Login)
* 🎯 Set fitness targets (water, weight, workout, sleep, steps)
* 📈 Track daily progress
* 📊 Visual progress indicators
* 👤 User profile
* 📱 Responsive design

---

## 🛠️ Tech Stack

### Frontend

* React (Vite)
* React Router
* Axios
* React Bootstrap
* CSS

### Backend

* Node.js
* Express.js
* MongoDB (Mongoose)
* JWT Authentication

---

## ⚙️ How It Works

1. User signs up / logs in
2. JWT token is generated and stored
3. User sets fitness targets
4. User updates daily progress
5. Data is stored in MongoDB
6. Progress is calculated and displayed visually
